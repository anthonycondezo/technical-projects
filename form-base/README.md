# Usage Notice
## Running Form-Base Server
To ensure that form-base server runs smoothly on your machine, please run the command "npm install" within project directory to install all module dependencies prior. Without module dependencies, form-base will NOT run. 
## Running Form-Base On Device
Once server is running, please sure that your install EXPO (from your app store) on your mobile device to access the form-base app. Else, please install the expo emulator on your machine to access on the app. 

## Emulator
The emulator can be download from: https://docs.expo.dev/get-started/set-up-your-environment/


### Additional Notes: 
Please note that the underlying postgres server may be inactive, causing limited functionality.
